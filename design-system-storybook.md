# CSS

## CSS의 약자: Cascading Style Sheet

> Cascading 의 사전적 뜻
> '폭포처럼, 계속되는, 연속적인

*윈도우 정렬 모드 중 Cascading으로 하면 차곡차곡 정렬되는 그런 형태를 말하는데, CSS는 '우선순위가 있는 스타일시트'라는 뜻입니다.*

CSS의 스타일 속성은 DOM 문서 트리에서 **상위 엘리먼트에 있는 스타일 속성이 하위 엘리먼트로 흘러내려서 상속**되어 적용됩니다.
